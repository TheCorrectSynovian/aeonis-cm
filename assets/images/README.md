# Assets Images Directory

This folder is for storing website images including:
- Header banner image
- In-game screenshots
- Custom spawn egg icons
- Copper Stalker egg texture
- Herobrine spawn egg texture

## Recommended Images:
- `banner.png` - Header banner image
- `copper-stalker-egg.png` - Copper Stalker spawn egg texture
- `herobrine-egg.png` - Herobrine spawn egg texture
- `wither-transform.png` - Wither transformation screenshot
- `dragon-flight.png` - Ender Dragon flight screenshot
- `herobrine-shrine.png` - Herobrine shrine screenshot

## Usage:
Images are referenced in `index.html` using `<img>` tags with the path `/assets/images/filename.png`
